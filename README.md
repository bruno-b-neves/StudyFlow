# StudyFlow

An intelligent language learning platform that combines spaced repetition, multimedia and artificial intelligence to personalize the learning experience.

## Features

- 📚 Flashcards
- 🔊 Audio support
- 🧠 Spaced repetition
- 📊 Learning statistics
- 🤖 AI-assisted learning
- ☁️ Cloud synchronization

## Tech Stack

### Frontend
- Nuxt 4
- Vue 3
- Bootstrap 5
- Pinia

### Backend
- Java 21
- Spring Boot 3
- Spring Security
- JPA/Hibernate

### Database
- MySQL
