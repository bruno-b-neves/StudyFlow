# Typography

## Objetivo

A tipografia define a hierarquia visual do StudyFlow e garante uma leitura confortável durante longas sessões de estudo.

Ela é baseada nos princípios do Material Design 3, porém adaptada para uma interface focada em aprendizagem.

# Guidelines

Utilizar no máximo três níveis tipográficos por tela.

Evitar excesso de tamanhos.

A hierarquia deve ser construída utilizando:

- tamanho
- peso
- espaçamento

Não utilizar cores para substituir hierarquia visual.
