# StudyFlow Design System

## Visão

O StudyFlow Design System é o conjunto de princípios, componentes e padrões visuais que garantem uma experiência consistente em toda a plataforma.

Inspirado pelo Material Design 3, o Design System do StudyFlow adapta suas diretrizes para um contexto de aprendizagem de idiomas, priorizando simplicidade, foco e acessibilidade.

Nosso objetivo não é reproduzir o Material Design, mas utilizá-lo como base para construir uma identidade própria.

---

## Objetivos

- Garantir consistência visual.
- Facilitar a evolução do produto.
- Reutilizar componentes.
- Melhorar a acessibilidade.
- Aumentar a produtividade da equipe.
- Reduzir retrabalho.

---

## Filosofia

Cada componente deve possuir uma única responsabilidade.

Cada tela deve ser construída a partir de componentes reutilizáveis.

Nenhuma decisão visual deve ser tomada diretamente durante o desenvolvimento.

Todo elemento utilizado na interface deve estar documentado neste Design System.
