# Colors

## Objetivo

O sistema de cores do StudyFlow define a identidade visual da aplicação e garante consistência entre todos os componentes.

As cores são inspiradas nos princípios do Material Design 3, mas adaptadas para uma experiência focada em aprendizado.

---

# Design Goals

As cores devem transmitir:

- Foco
- Calma
- Clareza
- Evolução
- Confiança

O usuário passará vários minutos estudando.

Portanto, evitamos cores extremamente saturadas.
