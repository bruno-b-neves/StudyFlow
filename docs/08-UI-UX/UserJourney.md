# Jornada do Usuário

"O que acontece desde o momento em que o usuário entra no StudyFlow até terminar uma sessão de estudo?"

Tela 1 — Login

Objetivo:

Permitir que um usuário existente entre na plataforma.

Ações:

* Entrar.
* Criar conta.
* Recuperar senha.

---

Tela 2 — Cadastro

Objetivo:

Criar um novo usuário.

Ações:

* Nome.
* E-mail.
* Senha.
* Confirmar senha.

---

Tela 3 — Dashboard

Essa já definimos.

Ela responde apenas uma pergunta:

"O que devo estudar hoje?"

O principal botão será:

▶ Iniciar Sessão

---

Tela 4 — Meus Decks

Aqui o usuário poderá:

* Criar Deck.
* Editar Deck.
* Excluir Deck.
* Ver quantidade de Flashcards.
* Ver progresso.
  
---

Tela 5 — Flashcards

Dentro de um Deck.

O usuário poderá:

* Criar Flashcard.
* Editar.
* Excluir.
* Pesquisar.
* Filtrar.
