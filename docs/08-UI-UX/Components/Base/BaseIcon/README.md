# BaseIcon

## Objetivo

O BaseIcon encapsula toda a biblioteca de ícones utilizada pelo StudyFlow.

Nenhum componente da aplicação deve utilizar diretamente a biblioteca Material Symbols.

Toda renderização de ícones deve ocorrer através do BaseIcon.

---

## Responsabilidade

Renderizar um ícone.

Controlar tamanho.

Controlar cor.

Controlar acessibilidade.

---

## Não deve

Conhecer regras de negócio.

Conhecer telas.

Conhecer Backend.

Conhecer Bootstrap.

---

## Biblioteca

Material Symbols Rounded

---

## Benefícios

Centraliza a implementação.

Permite trocar de biblioteca futuramente.

Mantém consistência visual.

Melhora acessibilidade.
